# Escape from Tarkov Remastered (SPT 4.x)

## Visão Geral

Escape from Tarkov Remastered é uma coleção cuidadosamente selecionada de mods para SPT, focada em:

* IA avançada
* Progressão equilibrada
* Imersão aprimorada
* Conteúdo lore-friendly
* Atmosfera expandida
* Experiência single-player de longo prazo

O objetivo não é transformar Tarkov em outro jogo, mas expandir a experiência original preservando sua identidade.

---

# Filosofia do Projeto

Este modpack foi criado seguindo alguns princípios:

* Preservar a experiência original do Tarkov
* Melhorar sistemas existentes ao invés de substituí-los
* Expandir conteúdo sem causar power creep excessivo
* Adicionar traders com identidade própria
* Melhorar a atmosfera sem comprometer performance
* Garantir compatibilidade para campanhas longas
